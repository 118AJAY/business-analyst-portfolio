📊 Supply Chain Analytics Project

🧩 Problem Statement
Businesses often lack real-time visibility into inventory levels, leading to overstocking or stockouts.

🎯 Objective
To design a data-driven dashboard that tracks inventory KPIs and supports decision-making.

🗂️ Data Source
Public dataset from Kaggle (Supply Chain / Inventory dataset)

About this file
Here is a dataset we collected from a Fashion and Beauty startup. The dataset is based on the supply chain of Makeup products. Below are all the features in the dataset:

Product Type
SKU
Price
Availability
Number of products sold
Revenue generated
Customer demographics
Stock levels
Lead times
Order quantities
Shipping times
Shipping carriers
Shipping costs
Supplier name
Location
Lead time
Production volumes
Manufacturing lead time
Manufacturing costs
Inspection results
Defect rates
Transportation modes
Routes
Costs

⚙️ Approach
- Data extraction using SQL
- Data cleaning and transformation
- KPI creation (Days of Inventory, Stock Levels)
- Dashboard creation using Power BI

📈 Expected Output
- Interactive dashboard
- Inventory insights

## 💡 Business Impact
- Improved inventory visibility
- Reduced manual reporting effort
